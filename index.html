4. Data Storage and Security

4.1 Local Storage

Most of your data is stored locally on your device using iOS secure storage mechanisms. This includes workout history, personal preferences, and app settings.

4.2 Cloud Storage

  • Firebase: User account information, workout data sync
  • iCloud: Optional backup of your data
  • Encryption: All data transmitted and stored is encrypted

4.3 Security Measures

  • End-to-end encryption for data transmission
  • Secure authentication using Firebase Auth
  • Regular security audits and updates
